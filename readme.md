🚀 SpendAI - Akıllı Harcama Yönetim Sistemi
SpendAI, geleneksel harcama takibi yöntemlerini yapay zeka ile birleştiren, kullanıcıların finansal verilerini analiz edip yönetebilmelerini sağlayan modern bir Full-Stack platformdur.

🛠️ Kullanılan Teknolojiler
Backend
Spring Boot: Uygulama mimarisi ve API katmanı.

Spring AI: Yapay zeka modelleri (Google Gemini / OpenAI) ile entegrasyon.

Spring Security & JWT: Stateless yetkilendirme ve rol tabanlı erişim yönetimi.

Spring Data JPA: PostgreSQL ile ilişkisel veri yönetimi.

Lombok: Verimlilik odaklı kod yazımı.

Frontend
HTML5 & CSS3: Modern ve kullanıcı dostu arayüz tasarımı.

Vanilla JavaScript: Gerçek zamanlı veri işleme ve API etkileşimi.

🌟 Öne Çıkan Özellikler
1. AI Destekli Akıllı Kayıt (Yeni!)
Kullanıcılar karmaşık formlar doldurmak yerine, günlük dilde harcamalarını yazar.

Örnek: "Bugün teknoloji mağazasından 2500 TL'ye kulaklık aldım."

AI İşleme: Yapay zeka bu cümleden tutarı (2500), kategoriyi (Teknoloji) ve açıklamayı otomatik olarak ayıklayıp veritabanına işler.

2. Gelişmiş Güvenlik ve Yetkilendirme
JWT tabanlı güvenli oturum yönetimi.

Admin Paneli: Tüm kullanıcıları listeleme ve yönetme yetkisi.

Kullanıcı Paneli: Kişisel harcama geçmişi ve AI asistanı.

3. Finansal Analiz
Kullanıcının harcama alışkanlıklarını analiz ederek bütçe tasarrufu ve kategori bazlı harcama raporları sunma hazırlığı.
